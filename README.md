# Flow Sprint Clock

ADHD-friendly focus timer built with Next.js for coding, project work, and study flow.

## Run locally

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run lint
npm test
npm run build
```

## Deploy

Push the repository to GitHub, then import it into Vercel as a standard Next.js project.

## Vercel

1. Import the GitHub repository into Vercel.
2. Accept the default Next.js settings.
3. Deploy without additional environment variables.
